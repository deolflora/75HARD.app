## What is 75 Hard?

75 days. 6 daily tasks. Miss one — you start over from day one. No excuses, no modifications, no compromises.

**Daily tasks:**
1. Two 45-minute workouts (one must be outdoors)
2. Drink 1 gallon of water
3. Read 10 pages of a non-fiction book
4. Follow a diet — no alcohol, no cheat meals
5. Take a daily progress photo
6. Complete the outdoor workout regardless of weather

## Features

- Daily goal tracking with one-tap toggles
- Streak system with XP and level progression
- 75-day visual grid showing your entire run at a glance
- Day celebration animations on completion
- Fully local — no account, no server, no tracking
- Persistent state via localStorage

## Tech Stack

- **React 19** — UI
- **TanStack Router** — file-based routing
- **TanStack Query** — data management
- **Tailwind CSS v4** — styling
- **Framer Motion** — animations
- **Vite** — build tool
- **TypeScript** — type safety
